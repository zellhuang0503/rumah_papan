# 驗收標準與 NFR (QA_Standards.md)

上線前的檢查清單。

## Acceptance Criteria

### Functional
- [ ] Switching language updates font-family immediately.
- [ ] All booking buttons redirect to correct Google Forms per language.
- [ ] Google Maps correctly displays tags from Figma variables.

### Performance
- [ ] Mobile LCP < 2.5s
- [ ] Image compression: WebP mandatory

### Constraints
- [ ] No layout changes allowed without approved Figma design.
- [ ] Strict adherence to HEX codes: `#F1592C`, `#242527`, `#EAE1D4`.
