## 📢 前言：關於這份手冊

這份手冊是為了讓你輕鬆管理網站內容而寫的。我們把網站的後台設計得很像填空題，只要把文字和照片放進去，網站就會自動更新。

**這份手冊會教你：**
1. 怎麼登入後台
2. 認識後台選單與網站頁面的對應關係
3. 【重點】怎麼新增、修改地圖上的景點
4. 怎麼發布最新消息（Facebook 貼文連結）
5. 怎麼管理故事誌文章（新增/修改/刪除）
6. 怎麼上傳與管理圖片
7. 怎麼修改其他頁面內容

## 📢 第三章：最新公告 — Facebook 貼文管理

這個功能讓你把 Facebook 粉專的貼文連結放到網站首頁的「最新消息」區塊。

### 3.1 新增一則公告

1. 點擊左側選單的 **「最新公告 (Announcement)」**
2. 點擊右上角的 **「+」** 或 **「Create new document」** 按鈕
3. 填寫以下欄位：

| 欄位名稱 | 說明 | 範例 |
|---------|------|------|
| **標題** | 公告的標題文字（必填） | 2026年春節營業時間公告 |
| **日期** | 公告日期（必填） | 2026.01.24 |
| **分類** | 選擇類型 | 最新 / 活動 / 公告 |
| **Facebook 貼文連結** | FB 原文網址（必填） | `https://www.facebook.com/...` |
| **顯示「最新」標籤** | 是否顯示紅色 NEW 標籤 | ✅ 開啟 |

4. 按下右下角的綠色 **「Publish」** 按鈕

### 3.2 取得 Facebook 貼文連結

1. 在 Facebook 找到你要分享的貼文
2. 點擊貼文右上角的 **「⋯」** 三個點
3. 選擇 **「複製連結」**
4. 回到後台，貼在 **「Facebook 貼文連結」** 欄位

### 3.3 修改公告

1. 在列表中點擊要修改的公告標題
2. 直接修改欄位內容
3. 按下 **「Publish」** 儲存

### 3.4 刪除公告

1. 在列表中找到要刪除的公告
2. 點擊右側的 **「⋯」** 三個點
3. 選擇 **「Delete」**
4. ⚠️ **注意：刪除後無法復原！**

## 🗺️ 第四章：觀光地圖景點管理

這是網站最核心的功能，讓遊客在地圖上找到好吃的肉骨茶和好玩的景點。

### 4.1 找到地圖設定

1. 點擊左側選單的 **「走進新村 (Village)」**
2. 點選列表中的文件（通常只有一個）
3. 往下滾動找到 **「觀光地圖頁面 (Map)」** 區塊
4. 展開後找到 **「景點列表 (Locations)」**

### 4.2 新增一個景點

滑到列表最下方，點擊 **「Add Item」** 按鈕，填寫以下欄位：

**📝 基本資料（必填）**
| 欄位 | 說明 | 範例 |
|-----|------|------|
| **ID** | 識別碼（英文小寫） | `ah-foong-bkt` |
| **名稱 (中文)** | 店家或景點名稱 | 阿火肉骨茶 |
| **Name (English)** | 英文名稱 | Ah Foong Bak Kut Teh |
| **副名稱/店名 (中文)** | 副標題 | 在地人推薦 50年老店 |
| **地址 (中文)** | 完整地址 | No.123, Jalan Besar... |
| **分類** | 選擇類型 | `food` / `temple` / `attraction` |

**🖼️ 圖片與特色**
| 欄位 | 說明 |
|-----|------|
| **圖片** | 點擊 Upload 上傳照片（建議正方形或 4:3） |
| **特色標題 (中文)** | 例如「招牌菜」 |
| **特色描述 (中文)** | 點擊 Add 新增多行，如「湯頭濃郁」、「肥瘦適中」 |
| **聯絡電話** | 店家電話 |
| **Google Map 連結** | 貼上 Google Maps 分享連結 |

**📍 設定地圖座標（大魔王關卡）**

這是最重要的一步！我們準備了 **「作弊小工具」** 給你：

1. 開啟網站前台，在網址後面加上 `?mode=admin`
   ```
   https://rumahpapanklang.com/village/map?mode=admin
   ```
2. 你會看到標題旁出現紅色的 **「🔧 Admin Mode」** 字樣
3. 在地圖上 **點擊你想放景點的位置**
4. 瀏覽器會彈出視窗顯示 X 和 Y 座標數字（已自動複製）
5. 回到後台填入：
   - **X 座標 (%)**: 填入第一個數字
   - **Y 座標 (%)**: 填入第二個數字

6. 按下 **「Publish」** 儲存

### 4.3 修改景點

1. 在景點列表中點擊要修改的項目
2. 直接編輯欄位
3. 按下 **「Publish」** 儲存

### 4.4 刪除景點

1. 點擊景點右側的 **「⋯」** 三個點
2. 選擇 **「Remove」**
3. 按下 **「Publish」** 確認刪除

## 📖 第五章：故事誌文章管理

想分享新村的活動、人物專訪或美食故事嗎？來這裡就對了。

### 5.1 新增故事文章

1. 點擊左側 **「故事誌 (Story)」**
2. 點擊右上角的 **「+」** 或 **「Create」** 按鈕
3. 填寫以下欄位：

| 欄位 | 說明 | 必填 |
|-----|------|:----:|
| **標題 (中文)** | 文章標題 | ✅ |
| **Title (English)** | 英文標題 | 選填 |
| **描述 (中文)** | 簡短介紹（約 2-3 行） | 選填 |
| **Slug** | 網址識別碼，點 Generate 自動產生 | 建議填 |
| **分類 (Category)** | 歷史/關於/交換/美食 | ✅ |
| **樣式 (Variant)** | 卡片顯示樣式 | 預設 Standard |
| **尺寸 (Size)** | 卡片大小 | 預設 Medium |
| **標籤 (Tags)** | 多個關鍵字標籤 | 選填 |
| **封面圖片** | 列表顯示的主圖 | 建議上傳 |
| **更多圖片** | 文章內的相簿圖片 | 選填 |
| **作者** | 撰文者姓名 | 選填 |
| **發布日期** | 文章發布時間 | 選填 |

4. 按下 **「Publish」** 發布

### 5.2 分類說明

| 分類 | 適合的文章類型 |
|-----|--------------|
| **歷史 (History)** | 新村歷史、老照片、回憶錄 |
| **關於 (About)** | 關於班厝的介紹、理念 |
| **交換 (Exchange)** | 換宿故事、志工分享 |
| **美食 (Food)** | 肉骨茶、在地小吃介紹 |

### 5.3 修改文章

1. 在文章列表中點擊標題
2. 直接修改內容
3. 按下 **「Publish」** 儲存

### 5.4 刪除文章

1. 點擊文章右側的 **「⋯」**
2. 選擇 **「Delete」**
3. ⚠️ **刪除後無法復原！**

## 🖼️ 第六章：圖片上傳通用指南

所有需要上傳圖片的地方，操作方式都一樣。

### 6.1 上傳步驟

1. 找到圖片欄位，點擊 **「Upload」** 按鈕
2. 選擇電腦中的圖片檔案
3. 等待上傳完成（會顯示預覽）

### 6.2 圖片規格建議

| 項目 | 建議 |
|-----|------|
| **檔案格式** | JPG、PNG、WebP |
| **檔案大小** | < 2MB（太大會上傳很慢） |
| **比例** | 視位置而定（見下表） |

| 使用位置 | 建議比例 |
|---------|---------|
| 景點圖片 | 正方形 1:1 或 4:3 |
| 故事封面 | 16:9 或 4:3 |
| 首頁輪播 | 16:9 寬幅 |
| 創辦人照片 | 正方形 1:1 |

### 6.3 焦點選取 (Hotspot)

有些圖片區塊支援 **Hotspot** 功能：
1. 上傳圖片後，點擊圖片
2. 會出現一個 **圓圈** 和 **方框**
3. 拖動圓圈到圖片的 **重要部位**（例如人臉）
4. 系統會確保裁切時不會切掉這個區域

### 6.4 替換圖片

1. 點擊已上傳的圖片
2. 點擊 **「Replace」** 或 **「Remove」**
3. 重新上傳新圖片

## 👥 第七章：後台帳號管理

如果需要多人共同管理後台，請依照以下步驟邀請新成員。

### 7.1 邀請新管理員

⚠️ 這一步 **只有專案擁有者 (Owner)** 才能操作：

1. 開啟 Sanity 管理頁面：`https://www.sanity.io/manage`
2. 登入後，點選專案名稱 (Project ID: `vm3p10fe`)
3. 點擊上方的 **「Members」** 標籤
4. 點擊 **「Add Member」** 按鈕
5. 輸入對方的 Gmail
6. 選擇權限角色：

| 角色 | 權限 | 適合誰 |
|-----|------|-------|
| **Administrator** | 最高權限，可刪除整個網站 | 老闆本人 |
| **Editor** | 可編輯所有內容 | 【推薦】小編、員工 |
| **Viewer** | 只能看不能改 | 實習生、觀察者 |

7. 按下 **「Send Invitation」**

### 7.2 對方如何啟用？

1. 對方會在 Gmail 收到邀請信
2. 點擊信中連結
3. 選擇 **「Login with Google」** 登入
4. 完成！現在可以進入後台時

## 💡 小撇步與注意事項

### ✅ 為什麼一定要按 Publish？

Sanity 後台是「即時儲存」的，但如果你不按綠色的 **Publish** 按鈕，這些改動只會存在草稿匣 (Draft) 裡，外面的遊客看不到。

**口訣：改完就要按 Publish！**

### ✅ 草稿 vs 已發布

- **藍色圓點**：有未發布的草稿變更
- **沒有圓點**：已發布，遊客看得到

### ✅ 圖片上傳太慢？

- 請確保圖片 < 2MB
- 網路不穩時請稍後再試
- 如果是直向照片，拍照時主體盡量放中間

### ✅ 遇到問題怎麼辦？

如果不小心刪錯資料，或網站顯示怪怪的：
1. 截圖保存錯誤畫面
2. 傳給技術團隊
3. 我們可以用 **History Restore** 幫你還原！
