【Figma 尺寸轉換規則（強制執行）】

- 我的 Figma 設計稿基準尺寸為：1920 × 1080
- 實際開發的 Desktop 基準尺寸為：1440px

你必須遵守以下轉換規則：

1. 所有來自 Figma（含 Dev Mode）的尺寸、spacing、position
   一律視為「1920 基準」，不得直接使用

2. 在實作前，必須先將所有 1920 基準尺寸
   等比例轉換為 1440 基準後，才能寫入程式碼

3. 轉換比例固定為：
   - scale = 1440 / 1920 = 0.75

4. 換算後規則：
   - width / margin / padding / gap / position
     → 一律乘以 0.75
   - font-size / line-height
     → 優先對齊 Tailwind 設計階梯（就近取值）
   - border-radius / blur / shadow
     → 視為視覺比例，同樣套用 0.75

5. 若我提供的數值已明確標示為「1440 基準」
   則不得再次縮放
